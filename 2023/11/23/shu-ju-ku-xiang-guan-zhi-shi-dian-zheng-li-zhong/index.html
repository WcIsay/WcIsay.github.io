еёёз”Ёзҡ„иҒҡеҗҲеҮҪж•°жңүе“Әдә”з§Қпјҹ

count()гҖҒsum()гҖҒavg()гҖҒmin()гҖҒmax()

count()пјҡжұӮиЎЁзҡ„иЎҢж•°жҲ–иҖ…жҢҮе®ҡиЎЁдёӯжҹҗдёӘеҲ—зҡ„еҲ—еҖјиЎҢж•°пјҢnullеҖјиў«еҝҪз•Ҙ

select count(еӯ—ж®ө) from иЎЁеҗҚ --еҫ—еҲ°иҜҘеҲ—еҖјзҡ„йқһnullеҖјзҡ„иЎҢж•°

select count(*) from иЎЁеҗҚ  --з”ЁдәҺз»ҹи®Ўж•ҙдёӘиЎЁзҡ„иЎҢж•°гҖӮд»»дҪ•иЎҢпјҢеҸӘиҰҒжңүдёҖиЎҢйқһnullпјҢеҲҷж•ҙдёӘиЎЁзҡ„иЎҢж•°е°ұдјҡиў«з»ҹи®ЎдёҠгҖӮе…ЁдёәnullпјҲдёҚдјҡеҮәзҺ°иҜҘжғ…еҶөпјүеҲҷдёҚиў«з»ҹи®Ў

sum()пјҡиҝ”еӣһжҢҮе®ҡж•°жҚ®зҡ„е’ҢпјҢеҸӘиғҪз”ЁдәҺж•°еӯ—еҲ—пјҢnullеҖјиў«еҝҪз•Ҙ

select sum(salary) from emp   --жұӮжүҖжңүsalaryзҡ„жҖ»е’Ң

max()пјҡиҝ”еӣһдёҖеҲ—дёӯзҡ„жңҖеӨ§еҖјпјҢnullеҖјиў«еҝҪз•Ҙ

select max(column_name) from table_name

min()пјҡиҝ”еӣһдёҖеҲ—дёӯзҡ„жңҖе°ҸеҖјпјҢnullеҖјиў«еҝҪз•Ҙ

select max(column_name) from table_name

avg()пјҡиҝ”еӣһж•°еҖјеҲ—зҡ„е№іеқҮеҖјпјҢnullеҖјиў«еҝҪз•Ҙ
